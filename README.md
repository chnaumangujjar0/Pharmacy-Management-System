# 💊 Pharmacy Management System

A complete **Pharmacy Management System** developed using **C# Windows Forms** and **SQL Server**, designed to streamline pharmacy operations such as inventory management, billing, and reporting.

---

## 📌 Overview

This application helps pharmacists efficiently manage:
- Medicine inventory
- Sales and billing
- Stock tracking
- Expiry monitoring
- Reports generation

The system is built with a focus on **usability, performance, and real-world workflow simulation**.

---

## ✨ Features

### 🔐 Authentication System
- Secure login system
- Tracks login attempts
- Prevents unauthorized access

---

### 💊 Medicine Management
- Add new medicines
- Edit existing records
- Delete medicines
- Search functionality for quick access

---

### 📦 Stock Management
- Track available stock levels
- Maintain stock history
- Update stock in real-time

---

### 🧾 Sales & Billing System
- Add medicines to cart
- Generate bills
- Calculate totals automatically
- Handle multiple items in one transaction

---

### 📊 Reports & Analytics
- 5 different report tabs
- View sales data
- Monitor stock usage
- Export reports to CSV format

---

### 📅 Alerts & Notifications
- Low stock alerts
- Expiry date warnings
- Helps prevent losses and shortages

---

## 🛠️ Tech Stack

- **Frontend:** C# Windows Forms (.NET Framework)
- **Backend:** SQL Server Express
- **Database Access:** Microsoft.Data.SqlClient
